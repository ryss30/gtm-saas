Setup instructions
Claude
  1. Copy the MCP server URL above.
  2. Open Claude → Settings → Connectors → Add custom connector → paste the URL → click Add.
  3. Sign in when prompted (magic link).
  4. Turn Orcool on in your chat (the tools menu), then start writing narratives.
ChatGPT
  1. Copy the MCP server URL above.
  2. Open ChatGPT → Settings → Apps → Advanced settings → Create app → paste the URL → confirm.
  3. Sign in when prompted. Orcool tools are now available in your chats.
Cursor
  1. Copy the MCP server URL above.
  2. In Cursor: Settings → MCP → Add new server → paste the URL.
  3. Sign in when prompted. Orcool is live in your MCP calls.

Questions

The honest answers.

How do we handle influencer payouts?

Cost-per-action: you pay for reach and conversions the story earns, not a flat fee up front. You only put budget behind a narrative that already proved itself on your account.

Can we A/B test narratives before influencer launch?

That is the whole point. You post variants as reels or organic spots on your account first. Only the version that holds gets handed to influencers, so paid reach goes behind a proven story.

Do you have an influencer network pre-loaded?

You bring the relationships or we help source them. Orcool is the story and the brief. Any influencer can shoot from a clean brief. Cost-per-action sits on distribution you control.

What if our account is small?

Orcool grounds stories in category signals, not your follower count. A narrative that works is a narrative that works, regardless of your account size. Test cheap, then scale.

How is this different than just writing scripts?

Grounding in live signals (what works now), testing at zero cost on your own channel, then exporting a brief influencers can scale. Script plus proof plus distribution.
